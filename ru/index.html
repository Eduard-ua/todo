  • Воспользуйтесь фильтром, чтоб посмотреть все задачи, выполнение, невыполненные
  • Сортируйте задачи по приоритету кликом на стрелки "Приоритет"
  • Найдите необходимую задачу начав писать ее название в поле "Поиск"
  • Для добавления списка дел напишите текст в поле ввода и нажмите Ввод или “Добавить задачу"
  • Сдвиньте движок для выбора приоритета задачи
  • Нажмите на задачу чтоб отметить ее как выполненую
  • Чтобы удалить один пункт, наведите на него и нажмите на значок корзины
  • Чтобы удалить все списки дел, нажмите "Очистить"
  • Нажмите "Сохранить", чтобы сохранить список дел на потом
